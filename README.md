## Technologies I've used
- node.js
- npm
- Yarn
- React.js
- Gulp.js
- SASS

- Application boilerplate with **[create-react-app](https://github.com/facebook/create-react-app)**


## Dev dependencies
- fundamental-react
- gulp
- gulp-sass
- gulp-watch
- node-sass
- react
- react-dom
- react-scripts
- sass-loader
